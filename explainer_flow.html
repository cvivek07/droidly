MyViewModel.kt — StateFlow
val uiState = MutableStateFlow<UiState>(Loading) // Fragment — safe collection with lifecycle lifecycleScope.launch { repeatOnLifecycle(Lifecycle.State.STARTED) { viewModel.uiState.collect { state -> render(state) } } }
StateFlow
Always has a current value. The recommended LiveData replacement in new code
repeatOnLifecycle
Cancels collection when STOPPED, restarts on START — same safety as observe()
